# SSPanel_Checkin
 GitHub action for sspanel automatically checks in to get free traffic quota
 这是一个用于GitHub action 的 sspanel 自动签到的菜鸡脚本
# How to use
 Add secrets
 在Secrets中增加

 AIRPORTUR、USERNAME、USERPASSWD

 spilt with &&
 以&&作为分割

# Example

AIRPORTUR https://github.com&&https://github.com
USERNAME 123&&123
USERPASSWD 456&&456
